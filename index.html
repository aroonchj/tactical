⚙️ ตั้งค่าฐานข้อมูล (ครั้งแรกเท่านั้น)

แอปนี้ใช้ Firebase Firestore เพื่อให้เก็บข้อมูลผู้เล่นและแผนแท็กติก แล้วเปิดดู/แก้ไขได้จากหลายเครื่อง (มือถือโค้ช, มือถือผู้จัดการทีม ฯลฯ)

  1. ไปที่ console.firebase.google.com สร้างโปรเจกต์ใหม่ (หรือใช้โปรเจกต์ Firebase อื่นที่มีอยู่แล้วก็ได้)
  2. เปิดใช้ Firestore Database (โหมด test mode พอสำหรับเริ่มต้น)
  3. ไปที่ ⚙️ Project settings → เลื่อนลงหา "Your apps" → คัดลอกค่า firebaseConfig
  4. วางทั้งก้อนลงในช่องด้านล่างนี้

ค่านี้จะถูกบันทึกไว้ในเครื่อง/เบราว์เซอร์นี้เท่านั้น ต้องตั้งค่าซ้ำในแต่ละเครื่องที่ใช้ (ครั้งเดียวต่อเครื่อง) โดยใช้ config ชุดเดียวกัน
